Dasar Privasi untuk Aplikasi Zikr Counter

Aplikasi Zikr Counter komited untuk melindungi privasi penggunanya. Dasar Privasi ini menerangkan bagaimana kami mengumpul, menggunakan dan melindungi maklumat anda semasa anda menggunakan aplikasi kami.

Pengumpulan dan Penggunaan Maklumat

Aplikasi ini tidak mengumpul atau menyimpan sebarang data peribadi pengguna. Semua data kiraan disimpan secara tempatan pada peranti pengguna dan tidak dihantar atau disimpan secara luaran.

Perubahan kepada Dasar Privasi

Kami mungkin mengemas kini Dasar Privasi kami dari semasa ke semasa. Oleh itu, anda dinasihatkan untuk menyemak halaman ini secara berkala bagi sebarang perubahan. Kami akan memaklumkan anda mengenai sebarang perubahan dengan menyiarkan Dasar Privasi baharu di halaman ini.
